# habit_os

An open-source task tracker for serious builders who want to stay committed to their goals without getting pulled into distractions.

habit_os is built around a simple idea: your task tracker should not just store tasks, it should wake your consciousness back up when you drift into random reels, messages, or low-value distractions. Think of it as an Alarmy-style accountability system for focus.

## Why This Exists

Most task apps are easy to ignore. You write the task, close the app, and still lose time to distraction.

habit_os is made for people who are serious about building something and need a sharper reminder system around their commitments.

## Goals

- Help users commit to meaningful goals.
- Reduce distraction from reels, random messages, and low-value scrolling.
- Keep the experience simple, direct, and focused.
- Stay open source so other builders can learn from it, improve it, or adapt it.

## Tech Stack

- Flutter
- Dart

## Getting Started

Clone the project, install dependencies, and run the app:

```bash
flutter pub get
flutter run
```

To check the project:

```bash
flutter analyze
```
